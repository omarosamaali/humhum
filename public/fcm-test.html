VAPID Key Setup Instructions:

  1. Go to Firebase Console
  2. Select your project: hum-hum-partner
  3. Go to Project Settings (gear icon)
  4. Click on Cloud Messaging tab
  5. Scroll to Web Push certificates
  6. Click Generate Key Pair if needed
  7. Copy the Key pair
  8. Update public/js/firebase-config.js with your VAPID key
